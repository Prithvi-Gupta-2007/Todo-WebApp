# Todo Fullstack App

A fullstack task manager built using MERN stack.

## 🚀 Features
- Add tasks
- Delete tasks
- Mark complete

## ⚙️ Tech Stack
- React
- Node.js
- Express
- MongoDB

## 📁 Structure
/backend → server  
/frontend → client
